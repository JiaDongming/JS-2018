10. div+css布局
    画盒子
    关键词：盒子，位置

    margin
    padding
    float
    position

    学的是什么？
        从本质上来说，如何去摆放盒子(div)的位置

    
    盒子模型
        盒子属性： 
            宽度:width
            高度:height
            边框:border
            内边距:padding
            外边距:margin

        作用：用来摆放盒子/内容 的位置
        规律：margin,border,padding,都具有四边

        border-left
        border-right
        border-bottom
        border-top

        简写：border: top,right,bottom,left
    margin：
            如果上下两个盒子间隔都设置了margin(上面盒子设置了margin-bottom,下面盒子设置了margin-top)，那么以大的那个为基准。塌陷现象

            盒子居中：

            /*非常常用的一条代码，盒子居中 ，盒子必须设置宽高 */
                margin: 20px auto; 
            

    padding：    实现把里面盒子往右边移 

        方法1：
             /*把里面盒子往右边移  */
            margin-left:30px; 
        方法2：
         /*外部盒子属性：把里面盒子往右边移,但是会撑宽外部盒子  */
            padding-left: 30px;


        需要注意的问题：
            设置了padding以后，会把外层盒子整体撑大
            影响：整个网页的布局
        
        盒子的总宽度： width + padding-left/right+ border-left/right

11. 块级元素和行内元素

    块级元素：独占一行，可以设置宽高
        div h1 p li

    行内元素：一行内显示，不可以设置宽高
        span i a img input

   /*把行内元素转成块级元素，不在一行内显示，支持设置宽高  */
            display: block;

            /*把行内元素转成块级元素，在一行内显示，支持设置宽高  */
            display: inline-block;

    /*把块级元素转成行内元素,原本宽高失效  */
        div {
                width: 200px;
                height: 100px;
                background-color: blue;
                margin: 10px 0;

                /*把块级元素转成行内元素,原本宽高失效  */
                /* display: inline;  */

                /*设置是否显示 none表示不显示 */
                display: none;
            }

16. 定位:可以把元素摆放到任意位置
相对定位： position:relative
    参照物：根据谁来做这个定位的? 
           根据定位之前自身的初始位置作为参照物

           根据没有使用定位之前初始位置的偏移量
           left
           right
           top
           bottom

绝对定位: position:absolute 
        参照物：上级元素中有定位属性(position)的，并且找的是最近的那个定位属性的父亲
        也有left， right，top，bottom
    
    绝对定位查找参照物的顺序：先找父级元素，如果有定位属性（postion属性），则以该父级元素为参照物；
    如果没有，则往上继续查找，直到找到有position的定位属性的元素为止。如果一直没有找到，则以
    最外层的html元素为参照物进行偏移。

    应用场景：右上角红点等。


固定定位：position:fixed
    参照物：浏览器
    应用场景：滚动后，需要保持悬浮不动到，比如广告，顶部导航栏，底部蓝等等。
